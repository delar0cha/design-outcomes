Brett McMillin walks through a fictional but recognizable scenario at "Astra," a fast-shipping AI video product, where the canvas captured four frames while the build quietly grew to fourteen states. The piece is structured as a workflow lab, which means it shows the actual tools, the actual skills, and the actual artifacts produced at each step. Worth reading for the workflow alone.

The reframe worth lifting is the one about source of truth. Most teams still treat the canvas, the codebase, and the PRD as three competing artifacts that need to be reconciled. Brett's framing replaces that with a system of connected artifacts: the canvas writes to code, code writes back to canvas, an agent translates between them, and the designer's judgment moves through both.

A second observation worth registering: in code, every exploration costs engineering time. The canvas keeps exploration nearly free. That asymmetry is why design has always paid for itself, and this tooling makes the math visible.
